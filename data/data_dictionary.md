# Data Dictionary

| Column | Description |
|---------|-------------|
| ride_id | Unique ride identifier |
| rideable_type | Type of bicycle used |
| started_at | Ride start date and time |
| ended_at | Ride end date and time |
| start_station_name | Starting station name |
| start_station_id | Starting station ID |
| end_station_name | Ending station name |
| end_station_id | Ending station ID |
| start_lat | Starting latitude |
| start_lng | Starting longitude |
| end_lat | Ending latitude |
| end_lng | Ending longitude |
| member_casual | Rider type |
| ride_length | Ride duration |
| day_of_week | Day of the week (0–6) |
| month | Month number (1–12) |
